# Ames Housing: Exploratory Data Analysis & Regression

## Overview
This project explores the Ames housing dataset using pandas, seaborn, and statsmodels.
The analysis covers data cleaning, feature engineering, exploratory analysis, and
regression modelling, including diagnostic checks and log-transformed models.

## Methods
- Data cleaning and missing value handling
- Feature engineering (TotalSquareFeet)
- Outlier detection using IQR
- Correlation analysis
- Simple and multiple linear regression
- Model diagnostics and log-transformation

## Tools
- Python
- pandas, numpy
- seaborn, matplotlib
- statsmodels

## Files
- `Ames_EDA.ipynb`: Jupyter notebook
- `Ames_EDA.html`: Rendered HTML version
